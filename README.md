Pron is the main Pronpy language-stack repository. It keeps one small starter example for each language and framework the organization wants to support.

## Start

```bash
git clone https://github.com/pronpy/pron.git
cd pron
python -m http.server 4173
```

Open `http://localhost:4173`.

## Language Stack

Explore the full stack in [LANGUAGES.md](LANGUAGES.md). Examples live in [`stack/`](stack/), including HTML, CSS, JavaScript, TypeScript, Python, Java, C, C++, C#, PHP, Ruby, Swift, Kotlin, Go, Rust, Dart, SQL, Shell, React Native, Flutter, and more.
